# 面板与页面导航

当前版本的主工作区不再只是几个单一页面，而是一整套围绕“运行态、配置、分析、管理”组织的面板系统。

## 常驻页面

### 概览

- 当前账号的核心状态页。
- 能看到运行态、任务状态、实时日志、统计卡片和部分运行时操作。
- 适合日常巡检。

### 总览

- 偏数据分析视角。
- 更适合看经验、金币、偷菜等统计图表。

### 农场预览

- 展示当前账号的农场核心信息。
- 用于快速确认账号基本资料、等级和农场资产。

### 好友

- 查看好友列表和好友农场状态。
- 支持缓存结果展示、手动刷新、筛选与状态识别。

## 操作类页面

### 偷菜设置

- 专门管理好友过滤、作物过滤、白名单、黑名单和偷菜策略。
- 这是社交互动的精细控制区，不建议只靠总开关粗放管理。

### 策略流程

- 提供可视化流程编排能力。
- 可以把种植、施肥、收获等步骤按流程组织，而不只是启用单个开关。

## 资料与工具页面

### 作物图鉴

- 用于看作物收益、经验、净利润、等级需求等。
- 当前版本的推荐逻辑会更关注真实等级门槛与收益效率。

### 农场工具

- 提供图鉴、经验计算器、氪金计算器、变异计算器等外部工具入口。
- 适合做“计算”“查资料”和“快速对照”，不替代主面板中的真实运行数据。

## 管理页面

### 账号

- 多账号管理的核心页。
- 支持启动、停止、批量操作、模式切换、状态筛选和视图切换。

### 设置

- 当前账号配置与管理员高级能力的统一入口。

## 管理员专属页面

### 公告管理

- 管理系统公告、草稿和下发内容。

### 系统日志

- 查系统级事件、风控回退、好友链路问题和后台异常。

### 操作日志

- 查管理动作是谁做的、什么时候做的、结果如何。

### 用户

- 管理系统用户与卡密续期相关信息。

### 账号归属

- 管理账号属于谁、能由谁操作、不同管理员之间如何区分。

### 卡密

- 管理卡密生成、复制、分发与批量处理。

## 推荐认知顺序

1. 先理解“概览 + 账号 + 设置”是日常主路径。
2. 再把“好友 + 偷菜设置 + 策略流程”视为策略调优区。
3. 最后把“系统日志 + 操作日志 + 账号归属 + 卡密”视为管理员后台。
