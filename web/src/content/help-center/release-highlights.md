# 近期版本重点更新

当前帮助中心基于 `v4.5.25` 的整体能力整理。这里不再手工抄录每个版本，而是把帮助中心改成直接读取仓库根目录 `CHANGELOG.md` 的快速索引，让版本说明和文档中心保持更稳定的联动。

## 这页怎么用

- 想知道某个版本主要改了什么，直接看下方自动同步的版本索引。
- 想知道某次更新对实际操作有什么影响，再回到帮助中心对应模块阅读完整说明。
- 如果帮助中心和更新日志出现口径差异，以最近一次更新后的真实页面与部署脚本为准，并继续补齐文档。

## 这对帮助中心意味着什么

旧版帮助中心里的很多“默认建议”已经不再适合当前版本，例如：

- 运维主线不再是零散裸机脚本
- 高风险项不再适合长期常开
- 设置页已经不是单页堆叠结构
- 管理后台能力已经从附属工具变成正式模块
